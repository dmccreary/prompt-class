# Prompt Types and Model Parameters

## Summary

This chapter explores the various categories of prompts and the model parameters that influence AI output. Students will learn to distinguish between directive, interrogative, open-ended, closed-ended, and conversational prompts, selecting the right type for each task. The chapter also covers critical model settings such as temperature, top-p sampling, and frequency penalties, giving students fine-grained control over the creativity, consistency, and reproducibility of AI responses.
