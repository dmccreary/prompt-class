# Core Prompt Techniques

## Summary

This chapter teaches the foundational prompting techniques that form the backbone of effective prompt engineering. Students will master zero-shot, one-shot, and few-shot prompting, learning how to select and structure examples for optimal results. The chapter also introduces chain-of-thought prompting, step-by-step reasoning, and prompt chaining, enabling students to guide AI models through complex reasoning tasks and build reusable prompt templates.
