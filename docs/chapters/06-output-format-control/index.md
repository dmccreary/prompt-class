# Output Format Control

## Summary

This chapter teaches students how to direct AI models to produce output in specific, structured formats. From Markdown and JSON to CSV, XML, YAML, and HTML, students will learn to specify and enforce output schemas for a wide range of use cases. The chapter covers both technical formats for data interchange and human-readable formats such as tables, bullet points, numbered lists, and narrative prose, giving students complete control over how AI-generated content is organized and presented.
