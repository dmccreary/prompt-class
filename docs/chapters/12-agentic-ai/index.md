# Agentic AI

## Summary

This chapter introduces the rapidly evolving field of agentic AI, where language models go beyond simple question-answering to autonomously plan, execute, and monitor complex multi-step tasks. Students will learn about agent architectures, tool use, function calling, and the planning-execution-feedback loop that enables autonomous workflows. The chapter covers agent orchestration, skill libraries, multi-agent systems, and critical safety considerations including human-in-the-loop approval workflows and sandboxed execution. Students will gain practical skills in designing, building, and safely deploying AI agent systems.
